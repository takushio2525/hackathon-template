# tools — 補助スクリプトの置き場（例）

本体のアプリケーションコードには含めないが、**開発や評価・検証に使う
スクリプト**を置く場所。ログ集計・ベンチマーク・データ変換・図の生成など。

## 構成（サンプル）

| ディレクトリ | 用途の例 |
|---|---|
| `example_benchmark/` | 性能測定スクリプトの書き方サンプル（Python 想定） |
| `example_analysis/` | データ解析スクリプトの書き方サンプル（Python 想定） |

どちらもあくまで**置き方の例**。中身は空なので、必要になったら実装する。

## 使い方

各スクリプトのディレクトリに `README.md` と実行ファイルを置く。依存する
ライブラリは `requirements.txt` や `pyproject.toml` 等で管理するとよい。

## 不要な班は

このディレクトリごと削除してよい。あとで必要になったら作り直す。
