## 選択肢と比較（例）

| 形式 | 長所 | 短所 |
|---|---|---|
| JSON | 人間が読み書きしやすい / ツールが豊富 | サイズが大きい |
| CSV | 表計算ソフトで編集できる | 階層構造が表現しにくい |
| 独自バイナリ | サイズが小さい / 高速 | デバッグしづらい |
| YAML | 可読性が高い | パーサ依存・字下げに注意 |
