# 楽譜データフォーマット仕様

楽譜データの形式は議論で決まってから書く。

候補：
- JSON（人間が読み書きしやすい，サイズが大きめ）
- 独自バイナリ（効率的だがデバッグしにくい）
- MIDI（既存ツールとの互換性が高い）
